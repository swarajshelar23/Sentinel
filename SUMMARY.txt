📈 THREAT SCORING PIPELINE (6 STAGES)
─────────────────────────────────────────────────────────────────────────────────

┌─ STAGE 1: Safe File Heuristics ──────────────────────────────────────────────┐
│ If all criteria met (format + VT=0 + entropy<6.8 + no YARA):                │
│ → RETURN: SAFE (0/100) ✓ Auto-classification complete                      │
└─────────────────────────────────────────────────────────────────────────────┘
        ↓ (if not auto-safe)
┌─ STAGE 2: File Signature Validation ────────────────────────────────────────┐
│ Check: Extension matches file header                                        │
│ Mismatch → +25 points (e.g., .exe disguised as .pdf)                       │
└─────────────────────────────────────────────────────────────────────────────┘
        ↓
┌─ STAGE 3: Entropy Analysis ─────────────────────────────────────────────────┐
│ <6.5: Normal | 6.5-7.2: Moderate | >7.2: High (packed/encrypted)          │
│ Scoring: 0 to +30 points based on file type and entropy level              │
└─────────────────────────────────────────────────────────────────────────────┘
        ↓
┌─ STAGE 4: VirusTotal Consensus ─────────────────────────────────────────────┐
│ Requirement: >5 engines needed for "strong" consensus                       │
│ Scoring: +0 (none) to +35 points (strong consensus)                        │
└─────────────────────────────────────────────────────────────────────────────┘
        ↓
┌─ STAGE 5: AI Model (Calibrated) ────────────────────────────────────────────┐
│ Raw probability calibrated based on:                                        │
│ - File type (safe doc bonus: ×0.6)                                         │
│ - Indicator count (low: ×0.7, high: ×1.2)                                  │
│ Scoring: +0 to +35 points                                                  │
└─────────────────────────────────────────────────────────────────────────────┘
        ↓
┌─ STAGE 6: YARA Signature Matching ──────────────────────────────────────────┐
│ High-confidence signatures only (filters generic patterns)                  │
│ Scoring: +8-40 points for matching patterns                                │
└─────────────────────────────────────────────────────────────────────────────┘
        ↓
┌─ FINAL CLASSIFICATION ──────────────────────────────────────────────────────┐
│ Score   0-30:  SAFE            (Green)                                      │
│ Score  30-60:  SUSPICIOUS      (Yellow)                                     │
│ Score  60-80:  MALWARE LIKELY  (Red)                                        │
│ Score  80-100: HIGH RISK       (Dark Red)                                   │
└─────────────────────────────────────────────────────────────────────────────┘
